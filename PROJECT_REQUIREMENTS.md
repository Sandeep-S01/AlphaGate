# Automated Crypto Trading Platform

## Goal

Build a production-grade automated crypto trading system connected with Binance API.

The system must:

- collect real-time market data
- analyze market conditions
- generate trading signals
- manage risk
- execute trades automatically
- maintain complete audit history


## Users

Initial:

Single user


Future:

Multiple traders
Multiple exchange accounts


## MVP

Included:

- Binance integration
- BTC/USDT trading
- strategy engine
- risk management
- paper trading
- dashboard


Excluded initially:

- AI prediction
- multiple exchanges
- mobile apps
- social trading


## Non Functional Requirements

The system must be:

- scalable
- observable
- secure
- testable
- maintainable
